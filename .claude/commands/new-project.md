新しいプロジェクト「$ARGUMENTS」を開始します。

## 手順
1. /Users/tomohirok/Documents/Github/claude-works/$ARGUMENTS/ にフォルダを作成
2. 依頼者に要件を確認する質問リストを作成（最重要）:
   - ビジネス要件（期間、条件、金額、対象者、法的制約）
   - 技術仕様（仕組み、導線、既存システムとの連携）
   - デザイン要件（参考サイト、トーン、ブランドとの整合性）
3. 質問リストを依頼者に提示し、回答を待つ
4. プロジェクト構造を初期化:
   - `docs/README.md` を作成（目的、技術スタック、セットアップ手順、ディレクトリ構成）
   - `作業記録.md` を作成（担当者、仕様、進捗を記録）
   - 主要ディレクトリ作成時に各ディレクトリの `README.md` を配置
5. 必要に応じて .claude/launch.json にプレビューサーバーを追加
