## 決定事項

### 1. 商品名: Ledgea（レジア）
- **由来**: Ledger（台帳）+ Asia
- **決裁者**: CEO
- **却下案**: Fixar（修理の誤解リスク）、Capex（汎用的すぎ）、Assetree（信頼感不足）
- **担当CxO**: CMO（提案）→ CEO（決裁）

### 2. タグライン
- EN: "The ledger built for Asia's complexity."
- JA: 「アジアの複雑さを、ひとつの台帳に。」
- **担当CxO**: CMO（原案）→ CEO（JA微修正・決裁）

### 3. ロゴ
- モチーフ: 3層カード（台帳）+ 碇（アンカー）抽象化
- カラー: Teal #0D9488 基調、Dark #134E4A、Mid #5EEAD4、Light #CCFBF1
- フォント: Plus Jakarta Sans（ロゴタイプ）
- バリエーション: icon / full-horizontal / full-vertical × 通常・白 = 6点
- favicon: SVGアイコンを流用
- **担当CxO**: CVO

### 4. ターゲット戦略
- 最優先: 日系企業のASEAN現地法人（経理・管理部門）
- 参入順序: 日本 → タイ → インドネシア → ベトナム
- 構造的優位: クライアントサイド完結 → 限界費用ゼロ → Freemium必然
- **担当CxO**: CSO（戦略）、CMO（ターゲット定義）

### 5. ブランドガイドライン
- brand/brand-guidelines.md として策定（v1.0 Draft）
- 6章構成: アイデンティティ / カラー / タイポ / ロゴ使用ルール / 文化的配慮 / ライティング
- **担当CxO**: CVO（ブランドガーディアン委任）
